Remove Pre-installed Apps in Windows 10

Table of Contents
  • Download the Windows 10 Debloater Script
  • Remove all Bloatware
  • Customize the Blacklist

Remove the applications that you do not need! Remove that nasty Candy Crush Saga and Xbox apps!

Removing the pre-installed applications will clear up a little bit of space and bloat from your Windows 10 install. This is useful especially if you just use your PC to play games or to do some work.

Thankfully, Sycnex already made a PowerShell script for removing those useless Windows 10 applications and disabling telemetry as well.

Download the Windows 10 Debloater Script

These steps can only be done by using an administrator account. You will be prompted for an administrator password if you use a standard account.

  1. Open Windows PowerShell by right-clicking the Start button and selecting Windows PowerShell (Admin).
  2. Copy and paste this command: iex ((New-Object System.Net.WebClient).DownloadString('https://git.io/debloat')) in the PowerShell window and press Enter.

This will download the debloater script and open the Windows 10 Debloater window.

Remove all Bloatware

You can click Remove All Bloatware right away if you do not use any of the Windows 10 applications! This will also disable telemetry, stop Cortana from indexing your PC, install .NET Framework 3.5, stop Edge PDF takeover and many more.

Customize the Blacklist

You can customize what to remove and what to retain. Click that Customize Blacklist button. Uncheck whatever you need to retain.

Then click Save at the top of the page. Click Remove Bloatware With Customized Blacklist.

Conclusion

Windows 10 Debloater is a handy tool to have. Run the script whenever you do a feature update since updates reinstall the apps and undo the changes that were made by this script.

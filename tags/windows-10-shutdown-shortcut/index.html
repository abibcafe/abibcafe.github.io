Windows 10 Shutdown, Restart and Log Off Desktop Shortcuts

Normally, shutting down, restarting or logging off a PC while using Windows 10 takes 2-3 clicks. Make shut down, restart and log off visible on the desktop by creating shutdown, restart, and log off desktop shortcuts on Windows 10. One can pin those shortcuts to the taskbar for much more easier access. Just be careful not to click those while working.
